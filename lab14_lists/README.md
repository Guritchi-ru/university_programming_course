[main.c](./main.c)

Создана структура узла, состоящего из указателя на следующий узел.
В качестве данных была взята структура уровень/комната Roguelike игры:

```bash
struct room {
    char name[50];
    int level;
    int number;
    int resolution;
}
```

Написаны функции: по добавлению узла в конец односвязного списка и вывод списка на экран.
Был создать односвязный список из n элементов, заполненных случайными данными. N – вводится с
клавиатуры/аргументами main.
